## 🏠 01 — Homelabbing

This project is about converting my old ASUS Pro PC 
which was gifted by my father and unused for 3 years 
into a fully functional homelab server. The PC runs 
an Intel i5 9th Gen with 16GB Corsair Vengeance DDR4 
RAM and a new Crucial BX500 1TB SSD. The operating 
system of choice is Ubuntu Server and all services 
run inside Docker containers for clean isolation and 
easy management.

The goal is to self host everything I currently depend 
on big tech for — music streaming with Navidrome, 
file storage with Nextcloud, ad blocking with Pi-hole, 
password management with Vaultwarden and more. The 
server will be accessible from anywhere in the world  via Tailscale VPN. This is Phase 1 of a larger plan that eventually includes Proxmox 
virtualization, a Raspberry Pi backup server and a 
full private cluster. 😈
